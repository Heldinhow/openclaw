# ROLE-COLLAB-RULES.md

## Collaboration Rules

### Routing Rules

- @commander → Commander (default for no mention)
- @strategist → Strategist
- @engineer → Engineer
- @creator → Creator
- @planner → Planner

### Response Priority

1. Explicit @mention takes priority
2. Commander is default for messages without @mention
3. Complex requests may involve multiple agents

### Handoffs

- Commander coordinates handoffs
- Each agent should read their SOUL.md before responding
- Update GROUP_MEMORY.md after significant interactions
